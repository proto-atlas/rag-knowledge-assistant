## Checked Implementation Points

- Search validation errors are exposed with `role="alert"`.
- Search loading, empty, and no-answer states use status or alert roles.
- The answer panel sets `aria-busy` while the stream is active.
- The UI has a polite live region for state-change messages.
- The RAG flow detail uses `aria-live="polite"` when the active step changes.
- Citation controls are buttons rather than non-interactive text.
- Activating a citation moves focus to the matching source card.
- Source card focus movement is covered by E2E test:
  - `tests/e2e/top-page.spec.ts`
